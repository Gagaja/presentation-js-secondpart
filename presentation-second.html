Variables

Variables are used so that we do not write the same value more than once, and that we can monitor which object is being done even when its value changes.

We can be immediately given the value when we create them, and we can also create them with the value added later, but we also have more individual variables

There are global and local variables. Global variables are available throughout the program, while local variables are defined within the function and will not be available outside the function.

Global variables

var dates = new Date();
var hours = dates.getHours();
if (hours >= 5 && hours <10) { document.getElementById( "global-example").innerHTML="Good morning" ; }
else if (hours < 19 && hours> 9) { document.getElementById("global-example").innerHTML = "Good afternoon"; }
else { document.getElementById("global-example").innerHTML = "Good evening"; }

Math object's

Math is a built-in object that has properties and methods for mathematical constants and functions. Not a function object. Unlike the other global objects, Math is not a constructor.
All properties and methods of Math are static. You refer to the constant pi as Math.PI and you call the sine function as Math.sin(x), where x is the method's argument.
Constants are defined with the full precision of real numbers in JavaScript.

Math methods

  • Math.abs(); // Absolute value (returns a positive number regardless of whether it was sent positive or negative)
  • Math.round(); // rounding up the numbers
  • Math.pow(); // step numbering
  • Math.sqrt(); // square root of the number
  • Math.ceil(); // Round the number up
  • Math.floor(); // Round the number down
  • Math.min();// From the forwarded numbers, the one returned is the smallest
  • Math.max(); // From the forwarded numbers, returns the one that is the largest
  • Math.random(); // Random number from 0 to 1
  • Math.PI; // Returns the PI number
